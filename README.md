# 📊 Sales Performance Dashboard

## Overview
This project is an interactive Sales Performance Dashboard created using Microsoft Power BI.

## Features
- Total Sales
- Revenue
- Total Orders
- Total Quantity
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Sales by Product
- Interactive Filters (Region & Category)

## Tools Used
- Microsoft Power BI
- Data Visualization
- DAX
- Data Analysis
